### Install Nodejs and npm
1. sudo apt update 
2. sudo apt upgrade 
3. curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
4. sudo apt install nodejs
5. sudo apt install npm


### SQLite3
